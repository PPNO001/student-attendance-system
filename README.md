# ระบบเช็คชื่อนักเรียน

ต้นแบบเว็บแอปสำหรับครูทุกคนในโรงเรียน เปิดได้จาก `index.html` หรือ GitHub Pages โดยไม่ต้องติดตั้งโปรแกรมเพิ่ม

## สิ่งที่ทำได้

- เช็คชื่อนักเรียนรายห้อง รายคาบ และรายครู
- Dashboard ภาพรวมทั้งโรงเรียน
- เลือกสิทธิ์ผู้ใช้ เช่น แอดมิน ครูประจำวิชา ครูที่ปรึกษา ฝ่ายปกครอง และผู้บริหาร
- Admin Console สำหรับจัดการนักเรียน ครู รายวิชา ห้องเรียน ตารางสอน คาบเรียน และสิทธิ์ผู้ใช้
- หน้า Admin มีตารางค้นหา เพิ่ม แก้ไข ลบ ตัวอย่าง CSV และปุ่มนำเข้า CSV แบบต้นแบบ
- ถ้าบันทึก "มา" หลังเวลาเริ่มคาบ ระบบจะเปลี่ยนเป็น "สาย" อัตโนมัติ
- คาบ 1 เริ่ม 08:40 ถ้านักเรียนเข้าโรงเรียนหลังเวลานี้ ระบบบันทึกสายคาบ 1 ทันที
- ค้นหานักเรียนที่เจอนอกห้อง แล้วดูได้ว่าควรอยู่คาบไหน วิชาอะไร ห้องไหน และครูคนใดสอน
- สแกน QR Code/Barcode แบบจำลอง ทั้งรายคาบ ครูสแกนบัตร และประตูโรงเรียน
- บันทึกการพบตัวนอกห้อง พร้อมสถานที่ การดำเนินการ และผู้บันทึก
- ดูรายงานนักเรียนสาย ขาด โดด พบอยู่นอกห้อง และนักเรียนเสี่ยง
- หน้าจัดการข้อมูลพื้นฐานและตั้งค่าคาบเรียน
- ตัวอย่าง role permission และ schema ฐานข้อมูลสำหรับต่อยอด PHP/MySQL
- ส่งออกข้อมูลเช็คชื่อเป็น CSV

## ไฟล์หลัก

- `index.html` โครงหน้าเว็บ
- `styles.css` งานออกแบบและ responsive layout
- `app.js` ข้อมูลจำลองและตรรกะระบบ

## ต่อยอดเป็นระบบจริง

ควรเพิ่มฐานข้อมูลกลาง, ระบบล็อกอินจริง, QR scanner จากกล้อง, LINE Messaging API, Export PDF/Excel และระบบสิทธิ์เชิงลึกก่อนใช้งานกับข้อมูลนักเรียนจริง
