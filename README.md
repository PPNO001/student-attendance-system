# ระบบเช็คชื่อนักเรียน

ต้นแบบเว็บแอปสำหรับครูทุกคนในโรงเรียน เปิดได้จาก `index.html` โดยไม่ต้องติดตั้งโปรแกรมเพิ่ม

## สิ่งที่ทำได้

- เช็คชื่อนักเรียนรายห้อง รายคาบ และรายครู
- ถ้าบันทึก "มา" หลังเวลาเริ่มคาบ ระบบจะเปลี่ยนเป็น "สาย" อัตโนมัติ
- คาบ 1 เริ่ม 08:40 ถ้านักเรียนเข้าโรงเรียนหลังเวลานี้ ระบบบันทึกสายคาบ 1 ทันที
- ค้นหานักเรียนที่เจอนอกห้อง แล้วดูได้ว่าควรอยู่คาบไหน วิชาอะไร ห้องไหน และครูคนใดสอน
- บันทึกการพบตัวนอกห้อง พร้อมสถานที่ การดำเนินการ และผู้บันทึก
- ดูรายงานนักเรียนสาย ขาด/โดด และรายการพบตัวนอกห้อง
- ส่งออกข้อมูลเช็คชื่อเป็น CSV

## ไฟล์หลัก

- `index.html` โครงหน้าเว็บ
- `styles.css` งานออกแบบและ responsive layout
- `app.js` ข้อมูลจำลองและตรรกะระบบ

## ต่อยอดเป็นระบบจริง

ควรเพิ่มฐานข้อมูลกลาง, ระบบล็อกอินครู, สิทธิ์งานปกครอง, QR/บัตรนักเรียน, การแจ้งเตือนผู้ปกครอง และแดชบอร์ดผู้บริหาร
